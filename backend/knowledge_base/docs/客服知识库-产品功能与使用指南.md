# 产品功能与使用指南（企业知识库示例）

## 工单系统

- 工单状态流转：待分配 → 处理中 → 待客户确认 → 已解决 → 已关闭。
- 超过 48 小时未响应的工单自动升级至组长。
- 支持工单合并：同一客户就同一问题重复提交时，客服可将工单合并处理。

## 智能路由规则

- 系统根据客户等级、问题类型、客服技能标签自动分配工单。
- VIP 客户（年消费 ≥50 万）的工单优先分配给高级客服。
- 技术类问题自动路由至技术支持组，商务类问题路由至客户经理。

## 知识库搜索功能

- 客服在回复工单时可调用内嵌知识库搜索，系统自动推荐相关文档片段。
- 知识库支持全文检索和语义检索两种模式，默认使用混合检索。
- 知识条目支持版本管理，修改记录可追溯。

## 数据看板

- 客服主管可查看实时数据看板，包含：在线客服数、排队客户数、平均响应时长、满意度评分。
- 数据看板每 5 分钟自动刷新，支持按时间段筛选。
- 满意度低于 85% 时系统自动发出预警通知。

## 常见问答

- Q: 工单能否转给其他客服处理？
  - A: 可以，处理中的工单支持转派，但需填写转派原因并通知客户。
- Q: 如何批量关闭历史工单？
  - A: 仅组长及以上权限可使用批量关闭功能，操作前需导出备份。
